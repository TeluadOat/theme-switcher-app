# Theme Switcher App

A simple React application that allows users to switch between **Light Mode** and **Dark Mode**, built as a beginner-friendly project for learning:

- React fundamentals
- Context API for global state management
- Tailwind CSS for styling
- LocalStorage persistence (optional stretch goal)

---

## Features

- Toggle between **light** and **dark** themes.
- Global theme state using **Context API**.
- Responsive UI styled with **Tailwind CSS**.
- Theme choice can be saved in **localStorage** (optional).

---

## Tech Stack

- **React** (Vite)
- **Tailwind CSS**
- **JavaScript (ES6+)**
- **Context API**
